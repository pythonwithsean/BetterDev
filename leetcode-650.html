The PythonWithSean 650

650 curated LeetCode problems organized by topic, from Easy to Hard. Complete this list and you will be ready for any technical interview or competitive programming contest.

How To Use This List
  • Go in order by topic. Complete one topic before moving to the next.
  • Within each topic, do Easy first, then Medium, then Hard.
  • Time yourself: 15-20 min for Easy, 25-35 min for Medium, 40-50 min for Hard.
  • If stuck after the time limit, read the editorial, understand it, then redo the problem the next day without looking.
  • Track your progress. Print this page or keep a spreadsheet. Mark each problem as you complete it.
